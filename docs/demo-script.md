## 🎥 Recording Tips

### Screen Setup
- **Resolution**: 1920x1080 minimum
- **Terminal**: Dark theme, large font (14-16px)
- **Browser**: Clean tabs, bookmarks hidden
- **Mouse**: Highlight cursor for visibility

### Tools
- **Recording**: OBS Studio, Loom, or QuickTime
- **Editing**: DaVinci Resolve (free) or Descript
- **Audio**: Use a microphone, not laptop speakers

### Pacing
- Speak clearly and slowly
- Pause between sections
- Keep total under 3 minutes
- Edit out mistakes/long waits

### B-Roll Suggestions
- SAP Explorer agent page
- Solscan transaction details
- Code snippets (workflow.ts, payment.ts)
- Ace Data Cloud dashboard (if available)
